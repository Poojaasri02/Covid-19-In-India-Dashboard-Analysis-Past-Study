# Covid-19-In-India-Dashboard-Analysis-Past-Study
COVID-19 in India Dashboard Analysis
Developed an interactive Tableau dashboard to analyze COVID-19 trends across India using data visualization techniques. The dashboard provides insights into state-wise cases, deaths, testing statistics, vaccination progress, age-group distribution, gender analysis, and ICMR testing laboratory availability. It enables users to explore pandemic trends through maps, charts, and real-time analytical visualizations for data-driven decision-making.

Key Features
🗺️ State-wise COVID-19 cases and deaths analysis
💉 Vaccination progress by vaccine type
👥 Age-group and gender distribution insights
🧪 ICMR testing labs across states
📈 Interactive charts, maps, and trend analysis
📊 Built using Tableau for effective data visualization
